How this actually works

Why can't this push to Google directly?

Google has never adopted the IndexNow protocol and has no public, free push-indexing API for ordinary pages. Its own Indexing API is restricted to job postings and livestream pages — using it for anything else risks Google flagging the account, so this tool doesn't attempt that. The honest path for Google is the sitemap you already maintain, plus manual "Request indexing" clicks in Search Console.

How does the backlink discovery page help?

It places your backlink URLs as real links on a Telegraph page — a domain search engines already crawl frequently. That gives crawlers a path to discover pages they wouldn't otherwise find quickly. It's a genuine technique used across the indexing-tool industry, but it's a probability boost, not a guarantee, and it can't substitute for a real backlink's own quality.

Is my IndexNow key sensitive?

No — it's published in a public text file on your own domain by design, so search engines can read it. Anyone with your key could submit IndexNow notifications for your domain on your behalf, which at worst causes unwanted re-crawls, not data exposure.

What stops someone from spamming this tool?

Every request needs a passed Turnstile check, is rate-limited per IP, and — for your-own-site submissions — must pass live domain ownership verification before anything is sent to IndexNow.
